Lets mine SoftNote on a CPU!

Get started with the list below.

  1. CPU mining SoftNote is extremely easy.

  2. Anyone can mine a Low-Difficulty block on their home computer when a low-difficulty block is allowed. Each Block Reward is 50 softnote coins.

  3. A low-difficulty block is a special event that happens if it's been 70 minutes a block has not been mined. At that time the difficulty will be turned down to 1 and even a home computer could mine the block.


  4. While you can CPU mine SoftNote it is extremely inefficent to try to mine a full-difficulty block on your CPU. Thoes people who still want to mine on their CPU can watch the video Only CPU mine when a low difficulty block is allowed
    5. to run the included script that will wait until a low-difficulty block is allowed, start mining, and only keep mining if a block has not been found. After a block is found and the special event is over the difficulty will return to its competitive height and your miner will stop mining. During the "Time Remaining:" period its just waiting and not mining.

  5. To CPU mine SoftNote please follow the instructions for your operating system below. If you are using Windows Operating System then please follow the directions in the Windows section and the same for Linux and MacOS.

  6. To CPU mine low-difficulty blocks please follow the instructions at this link BASH Script to only CPU mine when a low difficulty block is allowed. The instructions are the same for all operating systems. Ubuntu users should run the script directly from the bash terminal without escaping the forward slashes.

  7. Tapping the little grey box at the bottom of the screen will take you back to the top of the page

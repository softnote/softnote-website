Welcome to the SoftNote Quick Start Guide!

Get started with the list below. See the slideshow at the bottom.

  1. The Softnote Developer mined and currently retains 45,226,503 softnotes.

  2. SoftNote is a new cryptocurrency and is made up of these 4 simple computer programs that are easy to use and install:
    • SoftNote Core, The Softnote Blockchain Validator/Manager.

    • Electrum Wallet for SoftNote, The SoftNote Lightweight Wallet.

    • ElectrumX for SoftNote, The Electrum Wallet Server which provides data and services to Electrum Wallet for SoftNote users.

    • The SoftNote Blockchain Explorer, which allows SoftNote Core users to browse the blockchains data such as transactions inside blocks, the networks hashrate and much more.
  3. All of the computer programs you will need to use SoftNote are provided free of charge and on www.GitHub.com/softnote

  4. To get started sending, receiving and storing SoftNotes securely you will need:
    • Any Laptop or Desktop computer. Windows, Linux and MacOS are all compatible. MacOS and any other flavor of linux besides Ubuntu 18.04 and higher will be more difficult to setup. Its not guaranteed that linux on chromebook will work. ChromeOS alone without some kind of linux enabled is absolutely not compatible.

    • Optionally, a webcam to scan QR codes. You can send receive addresses and unsigned transaction files just like any other attachment like a photo or a text file stored on a thumb/USB drive.

    • A USB/Thumb Drive to store your wallet backup. Preferably two seperate backups on two seperate USB drives. Without a backup of some kind you will not be able to recover your coins in the event your data is corrupted or destroyed. They will be "Cryptographically Lost Forever". Instructions are given on how to make a backup. Its very easy and takes 1 minute.
  5. Everything you need to know about SoftNote will be introduced and explained as you follow this Quick Start Guide to get started.

  6. Each section of the Quick Start Guide is located in the drop down "Home Menu" at the top of the page and will introduce you to a new piece of the SoftNote collection of software or a very important topic so it is important to start at the first section titled SoftNote Core and read through each section to fully understand how to use SoftNote.

  7. To just view the contents of the SoftNote blockchain you need to download, install and run SoftNote Core and the SoftNote Blockchain explorer.

  8. If you just want to securely store, send and receive SoftNotes using as little computer resources as possible; You only need to download, install and run Electrum Wallet for SoftNote which will get service automatically from electrum.softnote.org:54002 .

  9. Laptop and Desktop users can mine softnotes on their CPU during speical occasions called "Low-Difficulty". It only happens if its been 70 minutes and a block has not been mined. I provide a script that waits for the occasion so you dont have to. Learn more about it here.

  10. SoftNote supports mining on Central Processing Units (CPU), Graphics Processing Units (GPU) and Application Specific Integrated Ciruits (ASIC).
    • To mine SoftNotes please visit the Mining Pool Software and the CPU, GPU and ASIC mining sections in the Home Menu at the top of the page.
  11. To stay updated on the lastest SoftNote Software releases, upcoming releases and SoftNote news, please follow us on social media by clicking any of the social media buttons at the top of the page.

  12. Tapping the little grey box at the bottom of the screen will take you back to the top of the page

  13. Click the SoftNote Logo to return to this page.
